# Boufbot
A C# Discord bot to do some text recognition on fight results of taxcollectors fights.

## Stack
- NET 9
- NetCord
- SixLabors.ImageSharp
- Tesseract

## How to use
Just configure your token in the appsettings.json file and run it. \
After that you can publish a screenshot in a channel and the bot will reply with the list of detected names.
